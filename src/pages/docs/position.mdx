## Static

Use `static` to position an element according to the normal flow of the document.

Any offsets will be ignored and the element will not act as a position reference for absolutely positioned children.

```html fuchsia
<template preview>
  <div class="relative p-6 bg-fuchsia-200">
    <div class="static h-32 p-4 font-medium bg-fuchsia-300 text-fuchsia-600">
      <p>Static parent</p>
      <div class="absolute bottom-0 left-0 p-4 bg-fuchsia-500 text-fuchsia-50">
        <p>Absolute child</p>
      </div>
    </div>
  </div>
</template>

<div class="**static** ...">
  <p>Static parent</p>
  <div class="absolute bottom-0 left-0 ...">
    <p>Absolute child</p>
  </div>
</div>
```

## Relative

Use `relative` to position an element according to the normal flow of the document.

Offsets are calculated relative to the element's normal position and the element *will* act as a position reference for absolutely positioned children.

```html rose
<template preview>
  <div class="relative p-6 bg-rose-200">
    <div class="relative h-32 p-4 font-medium bg-rose-300 text-rose-600">
      <p>Relative parent</p>
      <div class="absolute bottom-0 left-0 p-4 bg-rose-500 text-rose-50">
        <p>Absolute child</p>
      </div>
    </div>
  </div>
</template>

<div class="**relative** ...">
  <p>Relative parent</p>
  <div class="absolute bottom-0 left-0 ...">
    <p>Absolute child</p>
  </div>
</div>
```

## Absolute

Use `absolute` to position an element *outside* of the normal flow of the document, causing neighboring elements to act as if the element doesn't exist.

Offsets are calculated relative to the nearest parent that has a position other than `static`, and the element *will* act as a position reference for other absolutely positioned children.

```html indigo
<template preview>
  <div class="space-y-6">
    <div>
      <p class="mb-2 text-sm text-indigo-900">With static positioning</p>
      <div class="relative px-6 pt-2 pb-6 font-medium bg-indigo-200">
        <p class="mb-2 text-indigo-600">Relative parent</p>
        <div class="static p-4 text-indigo-700 bg-indigo-300">
          <p class="mb-2">Static parent</p>
          <div class="static bottom-0 left-0 inline-block p-4 bg-indigo-500 text-indigo-50">
            <p>Static child</p>
          </div>
          <div class="inline-block p-4 text-indigo-600 bg-indigo-200">
            <p>Static sibling</p>
          </div>
        </div>
      </div>
    </div>

    <div>
      <p class="mb-2 text-sm text-indigo-900">With absolute positioning</p>
      <div class="relative px-6 pt-2 pb-6 font-medium bg-indigo-200">
        <p class="mb-2 text-indigo-600">Relative parent</p>
        <div class="static p-4 text-indigo-700 bg-indigo-300">
          <p class="mb-2">Static parent</p>
          <div class="absolute top-0 right-0 inline-block p-4 bg-indigo-500 text-indigo-50">
            <p>Absolute child</p>
          </div>
          <div class="inline-block p-4 text-indigo-600 bg-indigo-200">
            <p>Static sibling</p>
          </div>
        </div>
      </div>
    </div>
  </div>
</template>

<div class="static ...">
  <!-- Static parent -->
  <div class="static ..."><p>Static child</p></div>
  <div class="inline-block ..."><p>Static sibling</p></div>
  <!-- Static parent -->
  <div class="**absolute** ..."><p>Absolute child</p></div>
  <div class="inline-block ..."><p>Static sibling</p></div>
</div>

```

## Responsive

To change how an element is positioned only at a specific breakpoint, add a `{screen}:` prefix to any existing position utility. For example, adding the class `md:absolute` to an element would apply the `absolute` utility at medium screen sizes and above.

```html
<div class="relative h-32 ...">
  <div class="inset-x-0 bottom-0 relative **md:absolute**"></div>
</div>
```

For more information about Tailwind's responsive design features, check out the [Responsive Design](/docs/responsive-design) documentation.
